# 生成AI超入門：初心者エンジニアが今日から試せる3つのTips

こんにちは！これは**超ざっくり**作ったサンプル記事です。まだ駆け出しエンジニアのあなたが、生成AIをサクッと触り始めるための３ステップを紹介します。

---

## 1. ChatGPTでコードレビューを依頼してみる
```
# PRの説明を貼り付ける
# ChatGPT には「このコードの改善点を教えて」と聞くだけ！
```
- 修正候補が自然言語で返ってくる
- すぐに「なるほど」と学習できる

## 2. GitHub Copilotでテストコードを自動生成
1. VS Code に拡張機能を入れる  
2. `// Test:` とコメントを書く  
3. Tab を押すだけで雛形が出現！

> Copilot まかせでも 8 割ぐらいテストが書けることも。

## 3. AI に API ドキュメントを要約させる
公式 Doc が長すぎる？ なら ChatGPT に要約を頼もう。

```
「OpenAI Functions のドキュメントを初心者向けに箇条書きで要約して」
```

---

### まとめ
- **全部無料枠で試せる**ものばかり
- まずは「とりあえず聞いてみる」スタンスが大事
